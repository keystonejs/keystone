The [`document`](../apis/fields#document) field type is a highly customizable rich text editor that lets content creators quickly and easily edit content in your system.

It's built with [Slate](https://docs.slatejs.org/), stores your content text as JSON-structured data, and lets you to things like:

- Link to other records in your project
- Configure your own custom marks and annotations
- Insert custom blocks based on React Components
